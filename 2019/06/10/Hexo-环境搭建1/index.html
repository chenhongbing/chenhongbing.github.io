5.生成SSH添加到Github

配置

查看

创建SSH，一路回车

生成.ssh文件夹，该文件下包含两个文件，分别为id_rsa和id_rsa.pub，id_isa为私钥，id_rsa.pub为公钥，复制id_rsa.pub的内容。
完成这样的配置：Github 账户下setting -> SSH KEYS -> New SSH KEY ->复制id_rsa.pub内容。
